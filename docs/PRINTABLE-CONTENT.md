## Print-Ready Output Specifications

### Standard Formats

```text
Textbook:        8.5" x 11" (Letter) or A4
Workbook:        8.5" x 11" (Letter) or A4
Quick Reference: 8.5" x 11" trifold or 4" x 6" cards
Poster:          24" x 36" or A1
Flash Cards:     3" x 5" or 4" x 6"
```

### Output Options

```text
PDF:    High-resolution, print-ready with bleed marks
DOCX:   Editable in Microsoft Word
LaTeX:  For mathematical/scientific content
HTML:   For web-to-print services
EPUB:   For e-readers with print option
```

## 1. Textbooks

### 1.1 Complete Subject Textbook

**What it does:** Full textbook with chapters, illustrations, exercises, and glossary.

**Prompt:**
```text
Create a complete printable textbook on "Introduction to Chemistry" for 9th grade:

Structure:
- Cover page with title, author space, school logo placeholder
- Table of contents with page numbers
- 12 chapters covering full curriculum
- Each chapter includes:
  - Learning objectives box
  - Main content with headers and subheaders
  - Key term definitions in margins
  - Diagrams and illustrations (describe for AI generation)
  - "Did You Know?" boxes
  - Worked examples with step-by-step solutions
  - Practice problems (answers in back)
  - Chapter summary
  - Review questions
- Appendices:
  - Periodic table
  - Common formulas
  - Unit conversion tables
  - Glossary with pronunciations
  - Index
- Answer key (separate section)

Format: PDF, 8.5" x 11", ready for binding
Style: Clean, modern, plenty of white space
Colors: Print-friendly (avoid full bleeds)
```

**Chapter outline example:**
```text
Chapter 5: Chemical Bonding
5.1 Why Atoms Bond
5.2 Ionic Bonds
5.3 Covalent Bonds
5.4 Metallic Bonds
5.5 Bond Polarity
5.6 Molecular Geometry
Practice Problems
Chapter Summary
Review Questions
```

## Print Production Tips

### File Preparation

```text
Resolution: 300 DPI minimum for print
Color Mode: CMYK for color, Grayscale for B&W
Bleed: 0.125" (3mm) for full-bleed pages
Margins: 0.5" minimum for binding
File Format: PDF/X-1a for professional printing
```

### Binding Considerations

```text
Saddle Stitch (stapled): Up to 64 pages
Perfect Binding: 48+ pages
Spiral/Coil: Any page count, lies flat
Three-Ring: Any page count, updatable
```

### Paper Recommendations

```text
Text Pages: 60-80 lb text weight
Covers: 80-100 lb cover weight
Workbooks: 50-60 lb with good erasability
Cards: 80-100 lb cardstock
Posters: 80-100 lb glossy
```
