## 1. 3D Visualizations

### 1.1 Solar System Model

**What it does:** Interactive 3D solar system with orbiting planets, accurate relative sizes, and clickable info panels.

**Prompt:**
```text
Create a 3D solar system visualization where:
- Planets orbit the sun at different speeds
- Clicking a planet shows its information (mass, distance, moons)
- Users can zoom in/out and rotate the view
- Include asteroid belt and dwarf planets
- Add realistic textures and lighting
```

**What you get:**
- Animated orbital mechanics
- Planet cards with scientific data
- Camera controls for exploration
- Optional: real-time position based on current date

**Variations:**
- Exoplanet systems comparison
- Historical view (planets in 1969 for Moon landing)
- "What if" scenarios (what if Earth was closer to the Sun?)
